# momo_chat
* python版本 3.10.13

# 创建项目的虚拟环境
* 参考：https://docs.streamlit.io/get-started/installation/command-line
* 创建虚拟环境：python -m venv .venv
* 激活虚拟环境：source .venv/bin/activate
* 安装相关依赖：pip install xxxx
* 退出虚拟环境：deactivate

# 通过streamlit启动本项目
* 运行命令：streamlit run MoMoChat.py
